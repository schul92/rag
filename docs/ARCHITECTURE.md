## Overview

### Cost Optimization Tips

1. **Skip Cohere Reranking** â†’ Save $30/month
   - Use HuggingFace BGE (free) or skip reranking
   - RRF fusion is often good enough

2. **Reduce Claude Calls** â†’ Save $4.50/month
   - Improve hybrid search to reduce "no results" scenarios
   - Pre-generate smart responses for common failures

3. **Use Voyage API Efficiently**
   - Cache embeddings for repeated queries
   - Batch similar queries together
